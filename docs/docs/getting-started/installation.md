`puppeteer-utilz` is [hosted on npm](https://www.npmjs.com/package/puppeteer-utilz), so you can install it & get up and running with either npm or yarn 👇
### npm

```sh
npm install puppeteer-utilz
```

### yarn

```sh
yarn add puppeteer-utilz
```

👩🏻‍💻 Happy hacking!
